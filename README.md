## :fire: Setup

```shell
# set enviroment variables
# * COOKIE_SECRET
# * PORT
# * NETWORK
# * NIS_PROTO
# * NIS_ADDR
# * NIS_PORT
# * NEM_PRIVATE_KEY
# * NEM_XEM_MAX
# * NEM_XEM_MIN
# * NEM_XEM_OPT
# * ENOUGH_BALANCE
# * MAX_UNCONFIRMED
# * WAIT_HEIGHT
# * RECAPTCHA_CLIENT_SECRET
# * RECAPTCHA_SERVER_SECRET
# or see .env.development

# install packages
$ npm install

# start app
$ npm start

# or for development
$ npm run dev
```
